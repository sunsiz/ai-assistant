Advanced AI-powered translation and content writing assistant for multilingual WordPress websites with support for Islamic content.

== Description ==

AI Assistant for WordPress is a comprehensive plugin that provides AI-powered translation and content writing assistance specifically designed for multilingual websites. With built-in support for Islamic content and terminology, it's perfect for religious organizations, educational institutions, and multilingual content creators.

**Key Features:**

* **Multiple AI Providers**: Support for OpenAI (GPT-3.5, GPT-4), Anthropic (Claude 3), and Google (Gemini)
* **URL-Based Translation**: Extract and translate content from any website URL
* **Content Suggestions**: AI-powered content generation based on titles and topics
* **Content Analysis**: Analyze existing content for improvements and SEO optimization
* **Classic Editor Integration**: Seamless integration with WordPress classic editor
* **Cross-Domain Support**: Perfect for separate language sites architecture
* **Islamic Content Optimization**: Specialized prompts and terminology handling for Islamic content
* **Translation History**: Complete database storage and management of translations
* **Multi-Language Support**: 16+ languages including Arabic, Chinese, Turkish, Persian, and more

**Perfect For:**

* Religious organizations with multilingual websites
* Educational institutions
* Content creators managing multiple language sites
* Bloggers and publishers needing translation assistance
* SEO professionals optimizing multilingual content

**AI Models Supported:**

* OpenAI: GPT-3.5 Turbo, GPT-4, GPT-4 Turbo
* Anthropic: Claude 3 Haiku, Claude 3 Sonnet, Claude 3 Opus
* Google: Gemini Pro, Gemini Ultra

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to AI Assistant > Settings to configure your API keys.
4. Configure your preferred AI models and customize prompts as needed.
5. Use the AI Assistant meta box in post/page editing screens to access features.

== Frequently Asked Questions ==

= Which AI providers are supported? =

The plugin supports OpenAI, Anthropic, and Google AI providers. You can use any combination of these providers and switch between different models as needed.

= Does this work with Gutenberg? =

No, this plugin is specifically designed for the WordPress Classic Editor. It provides a meta box interface for all AI features.

= Can I translate content from other websites? =

Yes! The URL-based translation feature allows you to input any website URL, extract its content, and translate it using your chosen AI model.

= Is Islamic content handled specially? =

Yes, the plugin includes specialized prompts and terminology handling specifically designed for Islamic content to ensure accuracy and cultural sensitivity.

= How is translation history managed? =

All translations are stored in a custom database table with full history tracking, including source URLs, languages, models used, and timestamps.
